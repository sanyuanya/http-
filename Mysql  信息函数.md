###  Mysql  信息函数



| CONNECTION_ID  | 连接id                   |
| -------------- | ------------------------ |
| 例子：         | SELECT CONNECTION_ID     |
| DATABASE()     | 当前数据库               |
| 例子：         | SELECT DATATBASE();      |
| LAST_INSERT_ID | 最后插入记录的id号       |
| 例子：         | SELECT LAST_INSERT_ID(); |
| USER()         | 当前用户                 |
| 例子：         | SELECT USER();           |
| VERSION        | 当前版本                 |
| 例子：         | SELECT VERSION();        |
